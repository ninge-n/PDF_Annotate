# PDF Annotate
PDFに文字、円、直線をを書き込んで、再びPDFとしてダウンロードできるツールです。役所などの申請書のPDFをブラウザで開き、氏名・住所などを追加して再度PDF化したいことがあるため作成しました。

## 使い方

1. `src/PDF Annotate.html` をブラウザで開く
2. 「PDFを開く」から追記したいPDFファイルを選択する
3. 書き込みを行う
   - テキスト追加: ボタンを押してキャンバス上をクリックし、文字を入力。ボックスの幅に合わせて自動で折り返されます
   - 円: ボタンを押してドラッグで描画。該当項目に丸をつける用途を想定（塗りつぶしなし・内側はクリック素通し）
   - 直線: ボタンを押して始点から終点へドラッグ。「＝」ボタンで二重線に切り替えられ、記載内容の取り消し線に使えます
   - いずれもドラッグで移動、ハンドルでサイズ・向きの変更、ホバーで表示される×で削除
4. 「PDFを保存」で `edited.pdf` がダウンロードされる

## 対応しているPDF

- 通常のPDF
- 編集制限付きPDF（閲覧は自由で編集のみ制限されたもの）
- 閲覧パスワード付きPDF: 開くときにパスワードの入力用ダイアログが表示されます。

## 技術メモ

単一HTMLファイルで構成され、ビルドは不要です。

### 使用ライブラリ(CDN)

| ライブラリ | 用途 |
|---|---|
| [pdf.js](https://mozilla.github.io/pdf.js/) | PDFの画面表示 |
| [@cantoo/pdf-lib](https://github.com/cantoo-scribe/pdf-lib) | PDFへの書き込み・保存（暗号化PDFの復号に対応したpdf-libフォーク） |
| [@pdf-lib/fontkit](https://github.com/Hopding/fontkit) | 日本語フォントの埋め込み |
| [M PLUS 1p](https://fonts.google.com/specimen/M+PLUS+1p) | 書き込むテキストのフォント（画面・PDF共通、約1.7MB） |

### 実装メモ

- `embedFont` の `subset: true` は fontkit のバグで日本語グリフが欠けるため使用しない（フォント全体を埋め込むので出力PDFは約1MB増える）
- テキストの折り返しは、ブラウザが実際に折り返した位置を Range API で読み取り、その行分割どおりにPDFへ描画している（画面とPDFのWYSIWYGを保証するため）
- 本家 pdf-lib は暗号化PDFを読めない（`ignoreEncryption: true` は出力が壊れるため不可）。復号対応の @cantoo/pdf-lib を採用している

## 既知の制限

- 初回のフォント読み込み時（表示・保存それぞれ初回）に数秒かかることがあります
- 書き込んだ内容の保存・再編集（作業状態の保存）には対応していません。ダウンロードしたPDFに追記することは可能です
